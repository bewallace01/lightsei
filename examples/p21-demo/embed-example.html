What this page is

A minimal stand-in for the customer's own product page. The bottom-right chat bubble is the Lightsei widget, injected by a single <script> tag at the bottom of this file.

How to try it locally

  1. Make sure the Lightsei dashboard dev server is running on localhost:3000 (or update the script src below).
  2. Open this file in a browser: open examples/p21-demo/embed-example.html.
  3. The chat bubble appears in the bottom-right.
  4. Click it to open the iframe.

In a production embed the snippet src would be https://app.lightsei.com/widget.js and the data-workspace attribute would be the public id from the workspace's widget settings page (Phase 21.7).

Local note: with no real workspace configured for the public id below, the iframe shows a "no widget is configured" error. Use the public id from a real seeded workspace to see the bot reply flow once Phase 21.6 ships.

The snippet

This is the entire integration:

<script src="https://app.lightsei.com/widget.js"
        data-workspace="{your public id}"
        async></script>
